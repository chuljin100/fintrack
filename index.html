Features

이런 기능이 있어요

📲

SMS 자동 감지

카드 결제 문자, 은행 알림을 자동으로 읽어 거래를 기록합니다. 신한, KB, 카카오, 토스, 농협 등 주요 금융사를 지원합니다.

🧠

AI 카테고리 분류

스타벅스는 식비, 올리브영은 쇼핑. AI가 가맹점을 분석하여 식비, 교통, 쇼핑, 의료, 주거, 금융으로 자동 분류합니다.

💰

일일 예산 계산

월 예산과 고정비, 이번 달 지출을 종합해 "오늘 쓸 수 있는 돈"을 실시간으로 계산해 줍니다.

🚨

과소비 오버레이 경고

5만원 이상 결제 시 화면에 경고 오버레이를 띄워줍니다. 10만원 이상이면 더 강력한 경고가 나타납니다.

💬

AI 잔소리 알림

결제할 때마다 AI가 따끔한 잔소리 메시지를 보내줍니다. 유머러스하지만 효과적인 소비 억제 장치입니다.

📈

목표 저축 예측

과거 3개월 소비 패턴을 분석하여 목표 금액을 기간 내에 달성할 수 있는지 AI가 예측해 줍니다.

How it works

이렇게 작동해요

1

앱 설치 후 알림 권한 허용

APK를 설치하고 SMS 수신 권한과 알림 접근 권한을 허용해 주세요.

2

카드 결제하면 자동 기록

카드 결제 SMS가 오면 돈소리가 자동으로 감지하여 금액, 가맹점, 날짜를 파싱하고 저장합니다.

3

AI가 분류하고 잔소리

서버의 AI가 거래를 카테고리별로 분류하고, 과소비 시 잔소리 메시지와 오버레이 경고를 보내줍니다.

4

대시보드에서 한눈에 확인

오늘 쓸 수 있는 돈, 이번 달 지출 현황, 목표 저축 달성 여부를 대시보드에서 바로 확인하세요.

Install Guide

APK 설치 방법

Play 스토어 외부 앱이라 보안 설정을 변경해야 합니다.
아래 단계를 따라해 주세요. (1분이면 끝!)

1

APK 파일 다운로드

위의 다운로드 버튼을 눌러 APK 파일을 받아주세요. 브라우저에서 "유해한 파일" 경고가 뜨면 "계속 다운로드"를 눌러주세요.

2

출처를 알 수 없는 앱 허용

다운로드한 APK를 열면 "보안상의 이유로 설치할 수 없습니다" 메시지가 뜹니다. "설정"을 눌러주세요.

설정 > 앱 > 특별한 접근 > 출처를 알 수 없는 앱 설치
3

브라우저 앱 허용

사용 중인 브라우저(Chrome, Samsung Internet 등)를 찾아 "이 출처 허용"을 켜주세요.

4

설치 진행

다시 APK 파일을 열면 설치가 진행됩니다. Google Play 프로텍트 경고가 뜨면 "무시하고 설치"를 눌러주세요.

5

권한 허용

앱 실행 후 SMS 수신 권한과 알림 접근 권한을 허용해 주세요. 돈소리가 카드 결제 문자를 자동으로 읽기 위해 필요합니다.

🔒 보안 안내: 돈소리는 해커톤 프로젝트로, 현재 APK 직접 배포 중입니다. 추후 개선을 거쳐 Google Play 스토어에 정식 출시될 예정입니다.
개인정보는 기기 내 Room DB에 저장되며, 서버에는 카테고리 분류를 위한 최소 정보만 전송됩니다.
